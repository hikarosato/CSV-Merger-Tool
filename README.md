# CSV Merger Tool
Програма дозволяє переносити переклади зі старого `.csv` файлу у новий.

Написано для [UE Localizations Tool](https://github.com/hikarosato/UE4LocalizationsTool), підтримується **Drag & Drop**.

1. Відкрийте новий `.locres` та експортуйте в `.csv`.
2. Відкрийте старий `.locres` з перекладом та експортуйте в `.csv`.
3. Об’єднайте обидва `.csv` через **CSV Merger Tool**.
4. Імпортуйте створений `.csv` в новий `.locres` або в **Google Sheets/Crowdin** для подальшого перекладу.
